# Admin page for shop laptop
#### Deployed on https://shop-laptop-admin.vercel.app/
## Getting Started

## Installation

```bash
$ npm install 
# or
$ yarn add
# or 
$ pnpm install
```

#### Add enviroment variables in .env.local

```
NEXT_PUBLIC_CHECKAUTH_URL
NEXT_PUBLIC_SERVER_URL
```

### Then run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
